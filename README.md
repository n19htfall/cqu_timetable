## 使用方法

1. **选课界面**（链接：[选课管理 - 重庆大学 (cqu.edu.cn)](https://my.cqu.edu.cn/enroll/CourseStuSelectionList)）- **查看课表** - **下载Excel**

2. 将下载的Excel（文件名为：“**课表.xlsx**”）放在根目录下。

3. 安装需要的依赖

   ```
   pip install -r requirements.txt
   ```

4. 运行main.py
